# ✈️ Make Me Fly
**Discover your next adventure.**

A flight deal discovery app that finds the cheapest flights from any airport to destinations worldwide. Compare economy and business class, find the best dates to fly, and calculate upgrade value — all powered by live airline data.

## Features
- **Flexible date search** — pick a month and trip length, see the cheapest dates to fly for every destination
- **Fixed date search** — compare fares across 58 major hub airports worldwide
- **Business class upgrade analysis** — see the real cost multiplier of upgrading
- **Deal scoring** — historical price comparison (powered by Amadeus Price Analysis)
- **Google Flights booking links** — one click to book
- **Region filtering** — search North America, Europe, Asia-Pacific, Africa, Middle East, Mexico/Caribbean, South America
- **Rate limiting & security** — IP-based, session-based, and global API caps

## Built With
- **Claude Code** — AI-powered development (zero lines of code written by hand)
- **Amadeus GDS API** — live airline pricing data from 400+ airlines
- **Streamlit** — Python web framework
- **3 Diet Cokes** — fuel
