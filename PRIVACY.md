This is a prototype application built for personal use and portfolio demonstration.

- No user accounts are created
- No personal data is collected, stored, or shared
- Streamlit uses session cookies for basic functionality (session management and XSRF protection). No tracking or advertising cookies are used.
- No analytics or tracking are implemented
- Flight search queries are not logged by this application. However, search queries are sent to the **Amadeus GDS API**, a third-party data provider, which may log requests per their own privacy policy.
- Search results are cached in memory for up to 30 minutes to improve performance and reduce API usage. Cached data is not persisted to disk and is cleared when the application restarts.

This application does not sell, rent, or share any user information.
